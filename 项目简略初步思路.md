# 项目简略初步思路

springboot + vue

**项目全称**：数字人文视域下黄河流域（山东段）文学景观构建与教学应用研究

**项目目标**：为中小学、大学文学教学提供可视化工具，直观展示诗人人文背景



## front前端端口一（资料整合端口）

### 功能 

主要目的是整合数据库

需要注册登录，增删改查，导入批量数据（也就是下面主题包含相关数据）

### 主题

诗人相关内容，景点相关内容，诗词相关内容，事件相关内容

(类似于管理员端口，负责数据管理)
参考图一

## front2前端端口二 （沉浸式展示端口）

### 功能 形象化展示已录入的数据，定制前端页面，直接访问不用登录

   页面1 地图视图检索

简略描述：地图范围是黄河流域山东地段，地图可放大缩小拖放。汇出黄河附近流域图，添加具体景点地区标签

交互：点击标签缩略图展示景点及介绍，点击详情可查看景点区域关联诗人、诗词（可跳转详情页）。
   页面2 诗人生平简介 概览、详情

简略描述：教育用途，内容调用后端接口访问数据库。展示诗人详细信息（考虑配合地图绘制经典出行经历与诗词创作）（可跳转详情）

   页面3 具体诗词详情，配合注解

   简略描述：诗词可通过模糊查询、精准查询等找到具体诗词，也可其他方式跳转到详情页面，介绍创作背景，配合ai视频，朗读

   页面4 朝代与诗人、事件

   简略描述：依照朝代时间线 查看事件与涉及诗人，背景下创建的诗词歌赋，更好了解时间背景，
